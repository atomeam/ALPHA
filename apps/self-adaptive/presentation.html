Deployment

Get Started in Minutes

1
Install
pnpm add -D wrangler
2
Configure KV
wrangler kv:namespace create METRICS
3
Set Secrets
wrangler secret put SLACK_WEBHOOK
4
Deploy
pnpm deploy

Or run locally with pnpm dev

API Endpoints

Simple REST Interface

GET /api/health Health check
POST /api/metrics Ingest metrics
GET /api/metrics Get recent metrics
POST /api/assess Run assessment
GET /api/status System status
GET /api/actions List pending actions
POST /api/actions/:id/approve Approve action
